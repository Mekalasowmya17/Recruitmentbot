# AI Recruitment Bot

An AI-powered recruitment assistant using:

- Gemini AI
- LangGraph
- Streamlit

## Features

- Resume Screening
- Candidate Scoring
- Interview Question Generation
- Hiring Decision

## Run

```bash
streamlit run app.py
```
